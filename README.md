# Twitch Chat Filter

A customizable browser extension for filtering chat messages on Twitch.

## Installation

### Chrome/Chromium-based Browsers

1. **Open the Extensions page:**
   - Navigate to `chrome://extensions/` in your browser
   - Or go to Menu → More Tools → Extensions

2. **Enable Developer Mode:**
   - Toggle the **"Developer mode"** switch in the top-right corner

3. **Load the extension:**
   - Click **"Load unpacked"**
   - Navigate to the folder containing this extension
   - Select the folder and click **"Select Folder"**

4. **Verify installation:**
   - The extension should now appear in your extensions list
   - A Twitch Chat Filter icon will appear in your toolbar

### Firefox

1. **Open the Add-ons page:**
   - Navigate to `about:debugging#/runtime/this-firefox` in your browser
   - Or go to Menu → Add-ons and Themes, then click the gear icon → Debug Add-ons

2. **Load the extension:**
   - Click **"Load Temporary Add-on"**
   - Navigate to the folder containing this extension
   - Select the `manifest.json` file and click **"Open"**

3. **Verify installation:**
   - The extension should now be listed under "Temporary Extensions"
   - A Twitch Chat Filter icon will appear in your toolbar

## Usage

1. **Visit Twitch:**
   - Go to [twitch.tv](https://www.twitch.tv)
   - Open any channel with live chat

2. **Open Extension Popup:**
   - Click the Twitch Chat Filter icon in your toolbar
